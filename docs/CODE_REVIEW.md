## 发现的问题

### 1. 编译与调用一致性 ✅ 已修复
- **问题**: `TaskProvider.addTask` 与调用方参数不一致，导致构建失败。
- **状态**: 已修复（新增 `reminderTime` 支持并统一调用）。

### 2. 任务导入导出一致性 ✅ 已修复
- **问题**: 导出/替换导入使用过滤后列表，存在数据遗漏风险；导入会丢失字段。
- **状态**: 已修复（改为全量任务导出；导入支持 replace/merge 并保留原字段）。

### 3. 通知生命周期与权限 ✅ 已修复
- **问题**: 提醒调度与取消不完整，Android 13+ 通知权限未处理。
- **状态**: 已修复（新增权限申请、统一调度/取消、稳定通知ID）。

### 4. Android 发布签名 ⚠️ 待配置
- **问题**: Release 签名需提供 `android/key.properties` 与 keystore。
- **状态**: 代码已支持，需在本地配置后验证发布构建。

### 5. 自定义优先级模型语义 ⚠️ 待优化
- **问题**: 自定义优先级最终映射到内置枚举，语义区分有限。
- **状态**: 待后续版本重构数据模型。
